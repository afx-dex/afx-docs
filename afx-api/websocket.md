# WebSocket

Persistent connection for real-time orderbook, kline, ticker, trades, and account state updates.

| Environment | URL |
| ----------- | --- |
| Mainnet | `wss://ws10.afx.xyz/ws/dex` |
| Testnet | `wss://ws10-testnet.afx.xyz/ws/dex` |
